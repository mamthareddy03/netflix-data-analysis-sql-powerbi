# 🎬 Netflix Data Analysis (SQL + Power BI)

## 📌 Overview
This project analyzes Netflix Movies and TV Shows dataset using SQL for data analysis and Power BI for visualization.

## 🛠️ Tools Used
- SQL (Data Cleaning & Analysis)
- Power BI (Dashboard & Visualization)
- Excel (Data Preparation)

## 🔄 Workflow
1. Data cleaning and transformation using SQL
2. Querying and analysis to extract insights
3. Importing processed data into Power BI
4. Building interactive dashboard with slicers and KPI cards

## 📊 Dashboard Features
- KPI Cards: Total Titles, Movies, TV Shows
- Content Distribution (Pie Chart)
- Content Growth Over Time (Line Chart)
- Top Countries and Genres
- Interactive Filters (Type, Year, Country)

## 📷 Dashboard Preview
![Dashboard](dashboard.png)

## 📈 Key Insights
- Content increased significantly after 2015
- Movies dominate Netflix catalog (~95%)
- USA and India are top content producers
- Drama and international genres are most popular

## 💼 Business Use Case
This project demonstrates how SQL and Power BI can be combined to transform raw data into actionable business insights.
