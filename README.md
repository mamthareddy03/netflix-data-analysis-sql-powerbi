# 🎬 Netflix Data Analysis (SQL + Power BI)

## 📌 Overview
This project analyzes Netflix Movies and TV Shows dataset to generate business insights using SQL and Power BI.

## 🛠️ Tools Used
- SQL
- Power BI
- Excel

## 📊 Key Insights
- Majority content is Movies
- Content growth increased after 2015
- Top countries: USA, India
- Popular genres: Drama, International

## 📁 Files
- netflix_titles.csv → dataset
- analysis.sql → SQL queries

## 🚀 Business Use Case
This project simulates real-world business analysis where raw data is transformed into insights for decision-making.
